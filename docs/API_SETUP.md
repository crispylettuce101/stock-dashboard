# API Setup Guide

## Reddit API Setup

1. Go to https://www.reddit.com/prefs/apps
2. Click "Create App"
3. Select "script" type
4. Get client_id and client_secret
5. Add to .env file

## Twitter API Setup

1. Go to https://developer.twitter.com/
2. Apply for developer account
3. Create app
4. Get Bearer Token
5. Add to .env file

See main README for detailed instructions.
